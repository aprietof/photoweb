#PaperKeep

Responsive landing page with dynamic content and modals. Built with HTML5, CSS3, Bootstrap 3.3.7 and jQuery 3.1.

##Directory structure

```
├── css
│   └── style.css
├── img
├── index.html
└── js
    └── script.js
```
